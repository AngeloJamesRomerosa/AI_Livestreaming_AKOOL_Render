The server streams your AI output as MJPEG — paste the URL into OBS as a Media Source. Your browser tab can be minimized and the stream will keep running.

1OBS → + → Media Source → uncheck "Local file"
2Paste the URL above into the Input field → OK
3OBS toolbar → Start Virtual Camera
4In streaming site/app/web → select OBS Virtual Camera

Opens the AI output in a clean standalone window fed directly from the server. OBS captures it via Window Capture — browser tab can be minimized.

1Click Open Pop Out Window above
2OBS → + → Window Capture → select the popup window
3Right-click the source → Filters → add Crop/Pad to trim browser chrome (Top=70)
4OBS toolbar → Start Virtual Camera

Stream directly to any RTMP platform (YouTube, Twitch, Facebook, etc.) — no OBS needed. The server encodes your video + mic audio and pushes it straight to your streaming URL.

Your stream key is never stored — it is used only for this session.

Not streaming
1Copy your RTMP URL + stream key from YouTube/Twitch/Facebook Studio
2Paste them above and click Go Live
3Allow microphone access when prompted — audio is captured automatically
4Click Stop Streaming when done
